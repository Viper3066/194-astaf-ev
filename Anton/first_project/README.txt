# Управление базой данных футбольных матчей

Этот проект представляет собой веб-приложение на Flask для управления базой данных футбольных матчей и дивизионов. Приложение позволяет добавлять, удалять, искать и просматривать данные о матчах и дивизионах. Также в проекте присутствует графический интерфейс на основе `tkinter` для локального управления базой данных.

## Основные функции

- **Добавление дивизиона**: Ввод кода дивизиона, названия и страны.
- **Удаление дивизиона**: Удаление дивизиона по его коду.
- **Добавление матча**: Ввод данных о матче (дивизион, дата, команды, голы, результат, сезон).
- **Удаление матча**: Удаление матча по его параметрам (дивизион, дата, команды).
- **Поиск матчей по дате**: Поиск всех матчей, которые состоялись в указанную дату.
- **Просмотр данных**: Отображение списка всех дивизионов и матчей.

## Технологии

- **Python**: Основной язык программирования.
- **Flask**: Веб-фреймворк для создания приложения.
- **SQLAlchemy**: ORM для работы с базой данных.
- **SQLite**: База данных для хранения информации.
- **HTML/CSS**: Шаблоны и стили для веб-интерфейса.
- **Tkinter**: Графический интерфейс для локального управления базой данных.
